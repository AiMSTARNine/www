# Aplikasi Pemesanan Cafe (DECAFE) BY Faris AiMSTAR
Sebuah aplikasi yang saya bangun untuk mempermudah pemesanan makanan dan minuman pada cafe, restoran, rumah makan atau sejenisnya.

## Pemilik/admin (1)
    -Melihat laporan
    -Menambah/mengubah/menghapus daftar menu dan harga
    -Menambah/mengubah/menghapus user
## Kasir (2)
    -Melihat harga dan total harga pesanan
    -Konfirmasi pembayaran
    -Membuat bukti pembayaran
## Pelayan (3)
    -Melihat menu dan harga
    -Memilih menu
    -Melihat harga total pesanan
    -Membuat peasanan
    -Update pesanan
    -Hapus pesanan
## Dapur (4)
    -Menerima pesanan
    -Konfirmasi pesanan
    -Mengubah status pesanan "siap saji"
